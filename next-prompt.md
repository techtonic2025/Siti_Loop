A modern landing page for the Stitch Skills Demo project.

**DESIGN SYSTEM (REQUIRED):**
Use a modern, clean, and professional design language.
- Fonts: Inter, sans-serif.
- Colors: Primary #3b82f6, Background #f8fafc, Surface #ffffff, Text #1e293b.
- Components: Rounded corners, subtle shadows, plenty of whitespace.
- Layout: Responsive, mobile-first.
- Styling: Use inline styles or a simple <style> block in the head. Avoid external CSS dependencies unless necessary.

**Page Structure:**
1. **Hero Section**:
    - Large, bold headline: "Build Websites at the Speed of Thought"
    - Subheadline: "Experience the power of Stitch Skills and autonomous coding agents."
    - Call to Action button: "Get Started"
2. **Features Section**:
    - Grid of 3 cards highlighting key features:
        - "Autonomous Loop": Iterative building with minimal human intervention.
        - "Design Systems": Consistent UI/UX across all pages.
        - "Agent Skills": Extensible capabilities for your coding assistant.
3. **Footer**:
    - Simple footer with copyright and links.

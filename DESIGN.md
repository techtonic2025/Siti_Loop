# Design System

## 1. Typography
- **Primary Font**: Inter (Google Fonts)
- **Headings**: Bold, prominent, dark slate color (#1e293b)
- **Body**: Regular, readable, slate color (#475569)

## 2. Color Palette
- **Primary**: #3b82f6 (Blue 500)
- **Secondary**: #64748b (Slate 500)
- **Background**: #f8fafc (Slate 50)
- **Surface**: #ffffff (White)
- **Accent**: #22c55e (Green 500)

## 3. Components
- **Buttons**: Rounded corners (0.5rem), primary background, white text, hover effects.
- **Cards**: White background, slight shadow, rounded corners (0.75rem), padding (1.5rem).
- **Navigation**: Sticky header, glassmorphism effect, clear links.

## 4. Layout
- **Container**: Max-width 1200px, centered.
- **Grid**: 12-column grid system.
- **Spacing**: 8px baseline grid.

## 5. Aesthetics
- **Style**: Modern, clean, professional, "SaaS" look.
- **Effects**: Subtle shadows, micro-interactions on hover.

## 6. Design System Notes for Stitch Generation
(Copy this section into prompts)
Use a modern, clean, and professional design language.
- Fonts: Inter, sans-serif.
- Colors: Primary #3b82f6, Background #f8fafc, Surface #ffffff, Text #1e293b.
- Components: Rounded corners, subtle shadows, plenty of whitespace.
- Layout: Responsive, mobile-first.
- Styling: Use inline styles or a simple <style> block in the head. Avoid external CSS dependencies unless necessary.

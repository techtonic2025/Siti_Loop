# STRYDA Image Assets

## Directory Structure

```
images/
├── products/          # Product photography
│   ├── supplements/   # Supplement jars and containers
│   └── gear/         # Gym equipment and accessories
├── hero/             # Hero section images
└── lifestyle/        # Lifestyle and gym environment shots
```

## Brand Guidelines

### Color Palette
- **Primary**: Black (#000000)
- **Accent**: Neon Green (#00FF00 / #39FF14)
- **Background**: Dark Gray (#1a1a1a)

### Photography Style
- Ultra-realistic commercial quality
- Studio lighting with dramatic shadows
- Premium reflections on packaging
- Dark gym/performance lab aesthetic
- Cinematic contrast
- 4K quality

## Image Requirements
- Format: WebP (primary), JPG (fallback)
- Resolution: 2000x2000px minimum
- Aspect Ratio: 1:1 for product shots, 16:9 for lifestyle
- File naming: `product-name-variant.webp`
- Optimization: Compressed for web performance

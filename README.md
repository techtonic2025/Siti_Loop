# Siti Loop Projects

This repository contains multiple web development projects created with Vite, React, and other modern technologies.

## Project Structure

The repository is organized into the following sub-projects:

### 1. **StrydaApp**
A premium fitness brand ecommerce application.
- **Tech Stack**: React, TypeScript, Vite, Tailwind CSS.
- **Key Features**:
  - Ultra-realistic product pages.
  - Interactive product gallery with studio and lifestyle shots.
  - Cart functionality (demo).
  - Responsive design.

### 2. **Techtonic Pet Store**
An online pet store demo.
- **Tech Stack**: React, TypeScript, Vite.
- **Key Features**:
  - Product listing for pet supplies.
  - Categories: Food, Toys, Care, Accessories.
  - Mock product data integration.

### 3. **Stitch Assets**
Shared assets and resources used across different Stitch projects.

## Getting Started

To run any of the projects locally:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/Siti_Loop.git
    cd Siti_Loop
    ```

2.  **Navigate to the project directory**:
    ```bash
    cd StrydaApp
    # OR
    cd techtonic-pet-store
    ```

3.  **Install dependencies**:
    ```bash
    npm install
    ```

4.  **Start the development server**:
    ```bash
    npm run dev
    ```
